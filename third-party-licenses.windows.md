# mcp-go-github dependencies

The following open source dependencies are used to build the [scopweb/mcp-go-github][] GitHub Model Context Protocol Server.

Some packages may only be included on certain architectures or operating systems.

 - [golang.org/x/oauth2](https://pkg.go.dev/golang.org/x/oauth2) ([BSD-3-Clause](https://github.com/golang/oauth2/blob/master/LICENSE))
 - [github.com/davecgh/go-spew](https://pkg.go.dev/github.com/davecgh/go-spew) ([ISC](https://github.com/davecgh/go-spew/blob/master/LICENSE))
