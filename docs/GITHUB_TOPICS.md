# GitHub Topics Recommendations

## How to Add Topics

1. Go to repository on GitHub
2. Click **Settings** (or About section on main page)
3. Click **Edit** next to "About"
4. Add topics in the "Topics" field

## Current GitHub Topic Limits

- Maximum: 20 topics per repository
- Minimum recommended: 5-8 topics
- Optimal: 8-12 topics
